# Project Management Form using JsonPowerDB

## Description

This project is a Project Management Form developed using HTML, CSS, JavaScript, Bootstrap and JsonPowerDB.

The application is designed to manage project records using Project ID as the primary key. It allows users to add new project details, retrieve existing project information automatically, update records and reset the form whenever required.

The project demonstrates frontend development, form validation and database integration using JsonPowerDB.

### Database Details

- Database Name: COLLEGE-DB
- Relation Name: PROJECT-TABLE
- Primary Key: Project-ID

### Fields Used

- Project-ID
- Project-Name
- Assigned-To
- Assignment-Date
- Deadline

## Project Workflow

### New Record

1. Enter Project ID.
2. If the Project ID does not exist, Save and Reset buttons are enabled.
3. Enter project details.
4. Click Save.
5. Record is stored in JsonPowerDB.

### Existing Record

1. Enter Project ID.
2. Existing data is fetched automatically.
3. Update and Reset buttons become active.
4. Modify the required details.
5. Click Update.
6. Database record is updated successfully.
